# Requirements: Astro Typefolio

**Defined:** 2026-03-27
**Core Value:** Provide a professional, elegant, minimal, static-fast personal publishing template that follows Occam's razor.

## v1 Requirements

### Home

- [ ] **HOME-01**: Visitor can understand Typefolio's target audience and purpose from the homepage hero/introduction
- [ ] **HOME-02**: Visitor can open the project's GitHub repository directly from the homepage
- [ ] **HOME-03**: Visitor can discover pinned posts from the homepage without navigating elsewhere first
- [ ] **HOME-04**: Visitor can see current news/recent updates from the homepage
- [ ] **HOME-05**: Homepage no longer uses the current list-style self/site presentation block

### About

- [ ] **ABOUT-01**: Visitor can find the detailed list-style personal/site presentation on the About page instead of the homepage
- [ ] **ABOUT-02**: About page copy explains Typefolio's audience, positioning, and template goals using project-specific language
- [ ] **ABOUT-03**: About page no longer contains upstream template framing or outdated placeholder descriptions

### Blog

- [ ] **BLOG-01**: Visitor can read blog index and featured blog copy that describes Typefolio rather than the previous upstream template
- [ ] **BLOG-02**: Public-facing showcase articles no longer contain outdated test/demo framing where they represent the template publicly
- [ ] **BLOG-03**: The current Markdown Admonitions article is retitled and repositioned as a Typefolio blog-components showcase
- [ ] **BLOG-04**: The blog-components showcase demonstrates admonitions, GitHub card, Astro Icon, TLDR, and backlinks in a coherent Typefolio-specific narrative
- [ ] **BLOG-05**: Updated article titles and descriptions reflect Typefolio's current feature set and release positioning

### Tags

- [ ] **TAGS-01**: Tag-related pages use Typefolio-specific archive copy rather than inherited upstream/template wording
- [ ] **TAGS-02**: Public tag surfaces no longer foreground obsolete test or placeholder framing

### Consistency

- [ ] **CONS-01**: Public-facing home, about, blog, and tag surfaces contain no visible leftover upstream-template introduction copy
- [ ] **CONS-02**: Completing this content refresh leaves the template ready to be presented as the official v1 release

## v2 Requirements

### Future Features

## Out of Scope

| Feature | Reason |
|---------|--------|
| New functional features beyond current template capabilities | This milestone is a content and presentation alignment pass |
| New page types or major route additions | Existing routes are sufficient for the official v1 release |
| README TODO implementation work | Explicitly deferred until after the v1 content refresh ships |
